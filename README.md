# LPG-Price-Crisis
A Power BI dashboard analyzing the impact of rising LPG prices on household expenses, rent, and financial burden across cities and income groups in India.
## 📊 LPG Price Crisis Dashboard

This project presents an interactive Power BI dashboard that analyzes the impact of rising LPG prices on the cost of living across different cities and income groups in India.

The dashboard focuses on understanding how fuel price fluctuations influence key aspects of daily life such as household expenses, rental costs, transport, and overall financial burden.

### 🔍 Key Features
- LPG price trend analysis over time  
- Cost impact comparison (LPG, rent, household expenses)  
- City-wise expense distribution  
- Income group-wise financial impact analysis  
- Financial burden indicator based on expense-to-salary ratio  

### 💡 Insights
- Increasing LPG prices contribute to a rise in overall living expenses  
- Lower-income groups are disproportionately affected  
- Urban areas show higher cost-of-living pressure  
- Transport and food costs further amplify financial stress  

### 🛠️ Tools & Technologies
- Power BI  
- Microsoft Excel  
- DAX (Data Analysis Expressions)  

### 🎯 Objective
To provide a data-driven understanding of how essential commodity price changes affect affordability and financial stability in real-world scenarios.
